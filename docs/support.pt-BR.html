Precisa de ajuda?

1. A postagem falhou / algo se comporta de forma estranha

Use o botão “Diagnosticar” → “Reportar” dentro do popup. Este é o canal canônico. O relatório é redigido (handles, URLs, e-mails) e encaminhado ao nosso rastreador privado de issues, onde um PR de correção de seletores é aberto automaticamente e enviado como hot-fix a todos os usuários em poucos dias.

Por que usar o botão de relatório dentro do produto?

O fluxo de relatório do popup anexa automaticamente o estado final de redação, um snapshot DOM redigido e os resultados de auditoria de seletores — exatamente o necessário para corrigir rapidamente a quebra de seletores. PII é removida através de múltiplas camadas antes do envio.

2. Perguntas gerais / solicitações de recursos

Escreva para contact@komm64.com. Este é um canal privado, seguro para compartilhar capturas de tela / handles pessoais.

3. Perguntas sobre instalação / conta

O Tutti reutiliza seus logins de navegador existentes para cada rede social. Não há conta separada do Tutti nem etapa de login. Uma vez conectado a uma rede social no mesmo perfil do Chrome, o Tutti pode postar a partir de lá.

FAQ

P. Minha postagem falhou.

Isso geralmente significa que uma rede social mudou seu DOM. Use Diagnosticar → Reportar no popup. Um hot-fix de seletor normalmente é enviado em poucos dias via a atualização automática de selectors.json (atualização da extensão não é necessária).

P. O Tutti não lembra meu login.

O Tutti depende do login normal da rede no navegador. Faça login em cada rede no Chrome primeiro e, então, o Tutti pode postar em seu nome. O modo anônimo não funciona porque as sessões da rede não estão lá.

P. Os limites de caracteres diferem por rede — isso é um problema?

Não. As postagens longas que excedem o limite de caracteres de uma rede são automaticamente divididas em uma thread de continuação (no X, uma cadeia de respostas adequada; em outras redes, postagens sequenciais). As postagens dentro do limite passam como estão.

P. Disseram que minha imagem / vídeo é grande demais.

Imagens são redimensionadas automaticamente via Canvas para se ajustarem ao limite de tamanho de cada rede. Vídeos são comprimidos automaticamente com ffmpeg.wasm quando excedem o limite de uma rede. Se nem a compressão couber, você verá um erro — verifique primeiro se o arquivo é enviado pelo aplicativo nativo da rede.

P. O Tutti vê o conteúdo da minha postagem?

Não. O conteúdo das postagens nunca passa por um servidor de terceiros. O Tutti injeta diretamente na página web de redação de cada rede. Consulte nossa Política de privacidade para detalhes. A única exceção é quando você pressiona o botão de relatório dentro do produto, que envia informações de erro redigidas para nosso rastreador privado de issues.

P. Posso doar?

Ainda não — estamos trabalhando em um canal de apoio sem pressão. O maior custo contínuo é manter-se atualizado com as mudanças de UI de cada rede social; futuros fluxos de Ko-fi financiarão essa manutenção.
