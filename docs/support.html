Need help?

1. Posting failed / something behaves oddly

Use the “Diagnose” → “Report” button inside the popup. This is the canonical channel. The report is redacted (handles, URLs, emails) and forwarded to our private issue tracker, where a selector fix PR is opened automatically and pushed as a hot-fix to all users within days.

Why use the in-product Report button?

The popup's Report flow automatically attaches the final compose state, a redacted DOM snapshot, and selector audit results — exactly what's needed to fix selector breakage fast. PII is stripped through multiple layers before sending.

2. General questions / feature requests

Email us at contact@komm64.com. This is a private channel, safe to share screenshots / personal handles with.

3. Install / account questions

Tutti reuses your existing browser logins for each social network. There's no separate Tutti account or login step. Once you're logged in to a social network in the same Chrome profile, Tutti can post from there.

FAQ

Q. My post failed.

This usually means a social network changed its DOM. Use Diagnose → Report in the popup. A selector hot-fix usually ships within days via selectors.json auto-update (no extension update required).

Q. Tutti doesn't remember my login.

Tutti relies on the network's normal browser login. Log in to each network in Chrome first, then Tutti can post on your behalf. Incognito mode doesn't work because the network sessions aren't there.

Q. Character limits differ per network — is that a problem?

No. Long posts that exceed a network's character limit are automatically split into a continuation thread (on X, a proper reply chain; on other networks, sequential posts). Posts within the limit go through as-is.

Q. I'm told my image / video is too large.

Images are auto-resized via Canvas to fit each network's size limit. Videos are auto-compressed with ffmpeg.wasm when above a network's cap. If even compression can't fit, you'll see an error — please verify the file uploads via the network's native app first.

Q. Does Tutti see my post content?

No. Post content never passes through a third-party server. Tutti injects directly into each network's web compose page. See our Privacy Policy for details. The only exception is when you press the in-product Report button, which sends redacted error information to our private issue tracker.

Q. Can I donate?

Not yet — we're working on a low-pressure support channel. The biggest ongoing cost is keeping up with each social network's UI changes; future Ko-fi flows will fund that maintenance.
