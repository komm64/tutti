Нужна помощь?

1. Публикация не удалась / что-то ведёт себя странно

Используйте кнопку «Диагностика» → «Сообщить» внутри всплывающего окна. Это канонический канал. Отчёт редактируется (имена пользователей, URL, email) и пересылается в наш приватный трекер задач, где автоматически открывается PR с исправлением селекторов и в течение нескольких дней доставляется как hot-fix всем пользователям.

Почему стоит использовать кнопку отчёта внутри продукта?

Поток отчёта во всплывающем окне автоматически прикрепляет конечное состояние составления поста, отредактированный DOM-снимок и результаты аудита селекторов — именно то, что нужно, чтобы быстро исправить поломку селекторов. PII удаляется через несколько слоёв перед отправкой.

2. Общие вопросы / запросы функций

Напишите нам на contact@komm64.com. Это приватный канал, безопасный для обмена скриншотами / личными именами.

3. Вопросы по установке / аккаунту

Tutti повторно использует ваши существующие входы в браузер для каждой соцсети. Нет отдельного аккаунта Tutti или шага входа. Как только вы вошли в соцсеть в том же профиле Chrome, Tutti может публиковать оттуда.

FAQ

В. Моя публикация не удалась.

Это обычно означает, что соцсеть изменила свой DOM. Используйте Диагностика → Сообщить во всплывающем окне. Hot-fix селектора обычно доставляется в течение нескольких дней через автоматическое обновление selectors.json (обновление расширения не требуется).

В. Tutti не помнит мой вход.

Tutti полагается на обычный вход в сеть через браузер. Сначала войдите в каждую сеть в Chrome, тогда Tutti сможет публиковать от вашего имени. Режим инкогнито не работает, потому что сессий сети там нет.

В. Ограничения на символы различаются для каждой сети — это проблема?

Нет. Длинные посты, превышающие ограничение на символы сети, автоматически разбиваются на ветку-продолжение (на X — правильная цепочка ответов; в других сетях — последовательные посты). Посты в пределах ограничения проходят как есть.

В. Мне сообщают, что моё изображение / видео слишком большое.

Изображения автоматически изменяются по размеру через Canvas, чтобы соответствовать ограничению размера каждой сети. Видео автоматически сжимается с помощью ffmpeg.wasm, когда оно превышает ограничение сети. Если даже сжатие не подходит, вы увидите ошибку — пожалуйста, сначала проверьте, что файл загружается через нативное приложение сети.

В. Видит ли Tutti содержимое моих постов?

Нет. Содержимое постов никогда не проходит через сторонний сервер. Tutti внедряется напрямую в веб-страницу создания поста каждой сети. См. нашу Политику конфиденциальности для деталей. Единственное исключение — когда вы нажимаете кнопку отчёта внутри продукта, которая отправляет отредактированную информацию об ошибке в наш приватный трекер задач.

В. Могу ли я сделать пожертвование?

Пока нет — мы работаем над ненавязчивым каналом поддержки. Самая большая текущая стоимость — это успевать за изменениями UI каждой соцсети; будущие потоки Ko-fi будут финансировать это обслуживание.
