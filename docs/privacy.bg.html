Tutti — Политика за поверителност

Без събиране на данни от трети страни. Цялото съдържание на публикациите отива директно от вашия браузър към всяка SNS.

Начало GitHub Поддръжка

Последна актуализация: 2026-05-23 (v0.5.0)

1. Данни, които събираме и предаваме

Tutti няма бекенд сървър. Съдържанието на публикациите (текст, изображения, видео) никога не се предава на сървър на трета страна. То пътува директно от вашия браузър към всяка избрана от вас социална мрежа, точно както ако сами сте го въвели и изпратили там.

  • Текстът, изображенията и видеото на публикациите се предават само на избраните от вас SNS страници (X, Bluesky, Threads, Mastodon, Misskey, Tumblr, Pixiv, DeviantArt, Instagram, TikTok, YouTube) във вашия браузър.
  • Когато опционалният път през официалния API е активиран за Bluesky / Mastodon / Misskey, публикациите се изпращат директно от вашия браузър към собствения API сървър на тази SNS с използване на пълномощията, които сте предоставили в Настройките. Никакъв Tutti / разработчиков сървър не е намесен.
  • Не се събират аналитични, проследяващи или рекламни данни.
  • Tutti не чете и не съхранява вашите токени за автентикация, бисквитки или сесийни данни от SNS сайтовете.

Изключение — ръчни доклади за грешки (съгласие на кликване)

Когато (и само когато) натиснете бутона „Докладвай“, показан в диалог за грешка или диагностичен панел, следното се изпраща през Cloudflare Workers релей към частен GitHub issue tracker (komm64/tutti-issues, видим само за разработчика на Tutti):

  • Съобщението за грешка, версията на разширението, низа User-Agent на браузъра.
  • Последните 30 записа на вътрешния лог на разширението (нивото на лога, което сте конфигурирали; INFO по подразбиране). Съдържание на публикации, изображения, видео, данни за вход в SNS и API пълномощия никога не са включени.
  • Редактиран DOM моментен снимка на SNS таба, където е възникнала грешката, използвана за поправка на счупени селектори. Текстово съдържание, стойности на атрибути, пътища/заявки/фрагменти на URL, атрибути href/src и тагове <meta>/<link> се премахват. Полезният товар е ограничен до 8 KB. Вижте dom-snapshot.ts за реализацията.

Нищо не се изпраща, докато не натиснете бутона за докладване. Дестинацията е частна, видима само за разработчика на Tutti.

2. Локално съхранявани данни

Следното се съхранява само вътре във вашия браузър чрез API storage на Chrome. Tutti не предава нищо извън устройството.

  • Настройки (chrome.storage.sync): URL на инстанции Mastodon / Misskey, превключвател autoPost, режим на показване, URL за override на селектори и т.н. Може да се репликира на другите ви устройства чрез Chrome Sync, ако сте влезли в Chrome.
  • Чернови (chrome.storage.session): Текст на публикация в процес и прикачени медии (изображения / видео). Автоматично се изчистват при затваряне на браузъра.
  • Избрани SNS (chrome.storage.local): SNS, които сте отметнали като цели на публикация. Запазва се между сесии за удобство; само локално на устройството.
  • Последно видени имена на потребители (chrome.storage.local): Текущото име на потребител, влязъл във всяка SNS, показано в popup, за да можете да потвърдите кой акаунт ще получи публикацията. Само локално на устройството.
  • История на публикации (chrome.storage.local): Последните 20 публикации: първите 80 символа текст, целеви платформи, успех/неуспех по платформа и времеви печат. Само локално на устройството.
  • API пълномощия (chrome.storage.local): Само ако изрично сте активирали пътя през официалния API за Bluesky / Mastodon / Misskey: вашата Bluesky app password или достъпен токен за Mastodon / Misskey. Само локално на устройството; никога не се репликира чрез Chrome Sync; никога не се предава на никой разработчиков сървър. Може да се премахне от съветника на Настройки.
  • Кеш за override на селектори (chrome.storage.local): Данни за hot-fix на селектори, извлечени от URL, който сте конфигурирали в Настройки (по подразбиране: публичният selectors.json на GitHub Pages на Tutti).
  • Кеш за лимит на качване на видео (chrome.storage.local): Резултат от API сондажи за лимити на качване на видео по акаунт (напр. Bluesky). Само числови стойности; без пълномощия.

Всичко това може да бъде изтрито по всяко време чрез деинсталиране на разширението или изчистване на хранилището за разширения на Chrome.

3. Използвани разрешения

РазрешениеЦел
storageСъхранява гореспоменатите локални данни (настройки, чернови, история, избрани платформи и т.н.) вътре във вашия браузър.
offscreenИзпълнява ffmpeg.wasm в offscreen документ за компресиране / прекодиране на видеофайлове, когато източникът надхвърля лимита на размера на целевата SNS. Видеоданните се обработват само локално; Tutti не ги предава никъде.
sidePanelОпционален режим на показване на UI: показва формуляра за съставяне на публикация на Tutti в страничния панел на Chrome вместо в popup на действие, така че да остане отворен, докато превключвате между SNS табове. Активирате го от страницата с Опции; стойността по подразбиране е стандартният popup.
host_permissionsИнжектиране на content scripts във всеки SNS сайт (x.com / twitter.com / bsky.app / bsky.social / threads.net / threads.com / mastodon.social / misskey.io / tumblr.com / pixiv.net / tiktok.com / youtube.com / instagram.com / deviantart.com) за попълване на формуляра за съставяне със съдържанието на вашата публикация и изпращането ѝ от ваше име. Никакви данни не се четат от тези сайтове и не се изпращат другаде; content скриптът само записва собствения ви вход.
optional_host_permissions: https://*/*Изисква се по време на изпълнение само ако конфигурирате персонализиран URL на инстанция Mastodon или Misskey (различен от стойностите по подразбиране). Chrome показва стандартното подканване за разрешение в този момент.

4. Връзка с доставчиците на SNS

Tutti автоматизира уеб UI (или, опционално, публикува чрез официалния API) на 11-те поддържани мрежи. Това е неофициална функционалност; Tutti не е свързан с X, Bluesky, Meta (Threads / Instagram), Mastodon gGmbH, Misskey, Tumblr, Pixiv, DeviantArt, TikTok или YouTube. Продължавате да сте обвързани със собствените условия за използване на всяка услуга.
