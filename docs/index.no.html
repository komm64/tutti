Tutti

Én Chrome-utvidelse. Elleve sosiale nettverk. Null styr.

GitHub Støtte Personvernerklæring

Oversikt

Tutti sender det samme innlegget til 11 sosiale nettverk (X, Bluesky, Threads, Tumblr, Mastodon, Misskey, Pixiv, TikTok, YouTube Shorts, Instagram, DeviantArt) med ett enkelt klikk. Den har ingen backend; innleggsinnhold går aldri gjennom noen tredjepartsserver.

Personvern i én setning

Tutti har ingen backend-server. Innleggsinnhold (tekst, bilder, video) går direkte fra nettleseren din til hvert sosiale nettverk du velger — aldri via en server kontrollert av utvikleren.
