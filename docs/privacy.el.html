Tutti — Πολιτική απορρήτου

Καμία συλλογή δεδομένων από τρίτους. Όλο το περιεχόμενο των αναρτήσεων πηγαίνει απευθείας από τον περιηγητή σας σε κάθε SNS.

Αρχική GitHub Υποστήριξη

Τελευταία ενημέρωση: 2026-05-23 (v0.5.0)

1. Δεδομένα που συλλέγουμε και μεταδίδουμε

Το Tutti δεν έχει διακομιστή backend. Το περιεχόμενο των αναρτήσεων (κείμενο, εικόνες, βίντεο) δεν μεταδίδεται ποτέ σε κανέναν διακομιστή τρίτου. Ταξιδεύει απευθείας από τον περιηγητή σας σε κάθε κοινωνικό δίκτυο που επιλέγετε, ακριβώς σαν να το είχατε πληκτρολογήσει και υποβάλει εσείς εκεί.

  • Το κείμενο, οι εικόνες και το βίντεο των αναρτήσεων παραδίδονται μόνο στις σελίδες SNS που επιλέγετε (X, Bluesky, Threads, Mastodon, Misskey, Tumblr, Pixiv, DeviantArt, Instagram, TikTok, YouTube) μέσα στον περιηγητή σας.
  • Όταν η προαιρετική διαδρομή επίσημου API είναι ενεργοποιημένη για Bluesky / Mastodon / Misskey, οι αναρτήσεις αποστέλλονται απευθείας από τον περιηγητή σας στον δικό του διακομιστή API του εν λόγω SNS χρησιμοποιώντας τα διαπιστευτήρια που παρέχατε στις Ρυθμίσεις. Δεν εμπλέκεται κανένας διακομιστής Tutti / προγραμματιστή.
  • Δεν συλλέγονται δεδομένα αναλυτικών στοιχείων, παρακολούθησης ή διαφήμισης.
  • Το Tutti δεν διαβάζει ούτε αποθηκεύει τα διακριτικά ελέγχου ταυτότητας, τα cookies ή τα δεδομένα περιόδου σύνδεσής σας από τους ιστότοπους SNS.

Εξαίρεση — χειροκίνητες αναφορές σφαλμάτων (συναίνεση ανά κλικ)

Όταν (και μόνο όταν) πατάτε το κουμπί «Αναφορά» που εμφανίζεται σε διάλογο σφάλματος ή στον πίνακα διαγνωστικών, τα ακόλουθα αποστέλλονται μέσω αναμετάδοσης Cloudflare Workers σε έναν ιδιωτικό ιχνηλάτη ζητημάτων GitHub (komm64/tutti-issues, ορατό μόνο στον προγραμματιστή του Tutti):

  • Το μήνυμα σφάλματος, η έκδοση της επέκτασης, η συμβολοσειρά User-Agent του περιηγητή.
  • Οι τελευταίες 30 καταχωρήσεις του εσωτερικού log της επέκτασης (το επίπεδο log που έχετε ρυθμίσει· INFO προεπιλογή). Το περιεχόμενο των αναρτήσεων, οι εικόνες, το βίντεο, τα δεδομένα σύνδεσης SNS και τα διαπιστευτήρια API δεν περιλαμβάνονται ποτέ.
  • Ένα επεξεργασμένο στιγμιότυπο DOM της καρτέλας SNS όπου συνέβη το σφάλμα, που χρησιμοποιείται για την επισκευή κατεστραμμένων επιλογέων. Το περιεχόμενο κειμένου, οι τιμές χαρακτηριστικών, οι διαδρομές/ερωτήματα/τμήματα URL, τα χαρακτηριστικά href/src και οι ετικέτες <meta>/<link> αφαιρούνται. Το payload είναι περιορισμένο σε 8 KB. Δείτε το dom-snapshot.ts για την υλοποίηση.

Τίποτα δεν αποστέλλεται μέχρι να πατήσετε το κουμπί αναφοράς. Ο προορισμός είναι ιδιωτικός, ορατός μόνο στον προγραμματιστή του Tutti.

2. Τοπικά αποθηκευμένα δεδομένα

Τα ακόλουθα αποθηκεύονται μόνο μέσα στον περιηγητή σας μέσω του API storage του Chrome. Τίποτα δεν μεταδίδεται εκτός συσκευής από το Tutti.

  • Ρυθμίσεις (chrome.storage.sync): URL στιγμιοτύπων Mastodon / Misskey, διακόπτης autoPost, λειτουργία εμφάνισης, URL υπερβολής επιλογέων κ.λπ. Μπορεί να αναπαραχθεί στις άλλες συσκευές σας μέσω Chrome Sync εάν είστε συνδεδεμένος στο Chrome.
  • Πρόχειρα (chrome.storage.session): Κείμενο ανάρτησης σε εξέλιξη και επισυναπτόμενα μέσα (εικόνες / βίντεο). Διαγράφεται αυτόματα κατά την έξοδο από τον περιηγητή.
  • Επιλεγμένα SNS (chrome.storage.local): Τα SNS που έχετε επιλέξει ως στόχους δημοσίευσης. Διατηρείται μεταξύ συνεδριών για ευκολία· μόνο τοπικά στη συσκευή.
  • Τελευταία ονόματα χρηστών που εθεάθησαν (chrome.storage.local): Το όνομα χρήστη που είναι συνδεδεμένο αυτή τη στιγμή σε κάθε SNS, εμφανίζεται στο popup ώστε να μπορείτε να επιβεβαιώσετε ποιος λογαριασμός θα λάβει την ανάρτηση. Μόνο τοπικά στη συσκευή.
  • Ιστορικό αναρτήσεων (chrome.storage.local): Οι τελευταίες 20 αναρτήσεις: οι πρώτοι 80 χαρακτήρες κειμένου, οι πλατφόρμες-στόχοι, επιτυχία/αποτυχία ανά πλατφόρμα και χρονοσφραγίδα. Μόνο τοπικά στη συσκευή.
  • Διαπιστευτήρια API (chrome.storage.local): Μόνο αν έχετε ενεργοποιήσει ρητά τη διαδρομή επίσημου API για Bluesky / Mastodon / Misskey: το Bluesky app password σας ή το διακριτικό πρόσβασης Mastodon / Misskey. Μόνο τοπικά στη συσκευή· ποτέ δεν αναπαράγεται μέσω Chrome Sync· ποτέ δεν μεταδίδεται σε κανέναν διακομιστή προγραμματιστή. Αφαιρείται από τον οδηγό Ρυθμίσεων.
  • Cache υπερβολής επιλογέων (chrome.storage.local): Δεδομένα hot-fix επιλογέων που λαμβάνονται από το URL που έχετε ρυθμίσει στις Ρυθμίσεις (προεπιλογή: το δημόσιο selectors.json στις GitHub Pages του Tutti).
  • Cache ορίου μεταφόρτωσης βίντεο (chrome.storage.local): Αποτέλεσμα ελέγχων API για όρια μεταφόρτωσης βίντεο ανά λογαριασμό (π.χ. Bluesky). Μόνο αριθμητικές τιμές· χωρίς διαπιστευτήρια.

Όλα αυτά μπορούν να διαγραφούν ανά πάσα στιγμή με την απεγκατάσταση της επέκτασης ή την εκκαθάριση του χώρου αποθήκευσης επεκτάσεων του Chrome.

3. Χρησιμοποιούμενα δικαιώματα

ΔικαίωμαΣκοπός
storageΑποθήκευση των παραπάνω τοπικών δεδομένων (ρυθμίσεις, πρόχειρα, ιστορικό, επιλεγμένες πλατφόρμες κ.λπ.) μέσα στον περιηγητή σας.
offscreenΕκτέλεση του ffmpeg.wasm σε ένα έγγραφο offscreen για συμπίεση / επανακωδικοποίηση αρχείων βίντεο όταν η πηγή υπερβαίνει το όριο μεγέθους του SNS προορισμού. Τα δεδομένα βίντεο επεξεργάζονται μόνο τοπικά· δεν μεταδίδονται πουθενά από το Tutti.
sidePanelΠροαιρετική λειτουργία εμφάνισης UI: εμφανίζει τη φόρμα σύνθεσης του Tutti στον πλευρικό πίνακα του Chrome αντί του popup ενέργειας, ώστε να παραμένει ανοιχτή ενώ μεταβαίνετε μεταξύ καρτελών SNS. Επιλέγετε από τη σελίδα Επιλογών· η προεπιλογή είναι το τυπικό popup.
host_permissionsΈγχυση content scripts σε κάθε ιστότοπο SNS (x.com / twitter.com / bsky.app / bsky.social / threads.net / threads.com / mastodon.social / misskey.io / tumblr.com / pixiv.net / tiktok.com / youtube.com / instagram.com / deviantart.com) για να συμπληρώνει τη φόρμα σύνθεσης με το περιεχόμενο της ανάρτησής σας και να την υποβάλλει για λογαριασμό σας. Δεν διαβάζονται δεδομένα από αυτούς τους ιστότοπους και δεν αποστέλλονται αλλού· το content script γράφει μόνο τη δική σας είσοδο.
optional_host_permissions: https://*/*Ζητείται κατά την εκτέλεση μόνο εάν διαμορφώσετε ένα προσαρμοσμένο URL στιγμιοτύπου Mastodon ή Misskey (διαφορετικό από τις προεπιλογές). Το Chrome εμφανίζει την τυπική προτροπή δικαιωμάτων εκείνη τη στιγμή.

4. Σχέση με τους παρόχους SNS

Το Tutti αυτοματοποιεί το web UI (ή, προαιρετικά, δημοσιεύει μέσω του επίσημου API) των 11 υποστηριζόμενων δικτύων. Αυτή είναι ανεπίσημη λειτουργικότητα· το Tutti δεν συνδέεται με X, Bluesky, Meta (Threads / Instagram), Mastodon gGmbH, Misskey, Tumblr, Pixiv, DeviantArt, TikTok ή YouTube. Παραμένετε δεσμευμένος από τους όρους χρήσης κάθε υπηρεσίας.
