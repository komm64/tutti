Tutti

Uma extensão do Chrome. Onze redes sociais. Zero complicações.

GitHub Apoio Política de privacidade

Visão geral

O Tutti transmite a mesma publicação para 11 redes sociais (X, Bluesky, Threads, Tumblr, Mastodon, Misskey, Pixiv, TikTok, YouTube Shorts, Instagram, DeviantArt) com um único clique. Não tem backend; o conteúdo das publicações nunca passa por nenhum servidor de terceiros.

Privacidade numa só frase

O Tutti não tem servidor backend. O conteúdo das publicações (texto, imagens, vídeo) vai diretamente do seu navegador para cada rede social que selecionar — nunca através de qualquer servidor controlado pelo programador.
