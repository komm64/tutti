Precisa de ajuda?

1. A publicação falhou / algo se comporta de forma estranha

Utilize o botão “Diagnosticar” → “Relatar” dentro do popup. Este é o canal canónico. O relatório é redigido (handles, URLs, e-mails) e encaminhado ao nosso repositório privado de issues, onde um PR de correção de seletores é aberto automaticamente e enviado como hot-fix a todos os utilizadores em poucos dias.

Porquê utilizar o botão de relato dentro do produto?

O fluxo de relato do popup anexa automaticamente o estado final de composição, uma captura DOM redigida e os resultados de auditoria de seletores — exatamente o necessário para corrigir rapidamente seletores partidos. PII é removida através de várias camadas antes do envio.

2. Questões gerais / pedidos de funcionalidades

Envie-nos um e-mail para contact@komm64.com. Este é um canal privado, seguro para partilhar capturas de ecrã / handles pessoais.

3. Questões de instalação / conta

O Tutti reutiliza as suas sessões existentes no navegador para cada rede social. Não há conta Tutti separada nem passo de início de sessão. Depois de iniciar sessão numa rede social no mesmo perfil do Chrome, o Tutti pode publicar a partir daí.

FAQ

P. A minha publicação falhou.

Isto normalmente significa que uma rede social mudou o seu DOM. Utilize Diagnosticar → Relatar no popup. Um hot-fix de seletor normalmente é enviado em poucos dias via a atualização automática de selectors.json (não é necessária atualização da extensão).

P. O Tutti não se lembra do meu início de sessão.

O Tutti depende do início de sessão normal da rede no navegador. Inicie sessão em cada rede no Chrome primeiro e, depois, o Tutti pode publicar em seu nome. O modo anónimo não funciona porque as sessões da rede não estão lá.

P. Os limites de caracteres diferem por rede — é um problema?

Não. As publicações longas que excedem o limite de caracteres de uma rede são divididas automaticamente numa thread de continuação (no X, uma cadeia de respostas adequada; noutras redes, publicações sequenciais). As publicações dentro do limite passam tal como estão.

P. Dizem que a minha imagem / vídeo é demasiado grande.

As imagens são automaticamente redimensionadas via Canvas para se ajustarem ao limite de tamanho de cada rede. Os vídeos são comprimidos automaticamente com ffmpeg.wasm quando estão acima do limite de uma rede. Se mesmo a compressão não couber, verá um erro — verifique primeiro se o ficheiro é enviado pela app nativa da rede.

P. O Tutti vê o conteúdo da minha publicação?

Não. O conteúdo das publicações nunca passa por um servidor de terceiros. O Tutti injeta diretamente na página web de composição de cada rede. Consulte a nossa Política de privacidade para detalhes. A única exceção é quando carrega no botão de relato dentro do produto, que envia informação de erro redigida ao nosso repositório privado de issues.

P. Posso doar?

Ainda não — estamos a trabalhar num canal de apoio sem pressão. O maior custo contínuo é manter-se a par das alterações da UI de cada rede social; futuros fluxos Ko-fi financiarão essa manutenção.
