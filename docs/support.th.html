Tutti — การสนับสนุน

แหล่งข้อมูลสนับสนุนและคำถามที่พบบ่อย

หน้าแรก GitHub นโยบายความเป็นส่วนตัว

ต้องการความช่วยเหลือ?

1. การโพสต์ล้มเหลว / มีบางสิ่งทำงานผิดปกติ

ใช้ ปุ่ม «วินิจฉัย» → «รายงาน» ภายใน popup นี่คือช่องทางมาตรฐาน รายงานจะถูกแก้ไขข้อมูล (handles, URLs, อีเมล) และส่งต่อไปยังตัวติดตามประเด็นส่วนตัวของเรา ซึ่งจะเปิด PR แก้ไขตัวเลือกโดยอัตโนมัติและส่งเป็น hot-fix ให้ผู้ใช้ทุกคนภายในไม่กี่วัน

ทำไมต้องใช้ปุ่มรายงานในผลิตภัณฑ์?

ขั้นตอนการรายงานของ popup จะแนบสถานะการสร้างโพสต์สุดท้าย, ภาพรวม DOM ที่แก้ไขแล้ว, และผลการตรวจสอบตัวเลือกโดยอัตโนมัติ — แน่นอนสิ่งที่จำเป็นในการแก้ไขการแตกหักของตัวเลือกอย่างรวดเร็ว PII ถูกลบผ่านหลายชั้นก่อนการส่ง

2. คำถามทั่วไป / คำขอคุณสมบัติ

ส่งอีเมลถึงเราที่ contact@komm64.com นี่เป็นช่องทางส่วนตัว ปลอดภัยสำหรับการแชร์ภาพหน้าจอ / handles ส่วนตัว

3. คำถามเกี่ยวกับการติดตั้ง / บัญชี

Tutti นำการล็อกอินเบราว์เซอร์ที่มีอยู่ของคุณกลับมาใช้ใหม่ สำหรับแต่ละเครือข่ายสังคม ไม่มีบัญชี Tutti แยกหรือขั้นตอนการเข้าสู่ระบบ เมื่อคุณเข้าสู่ระบบเครือข่ายสังคมในโปรไฟล์ Chrome เดียวกัน Tutti สามารถโพสต์จากที่นั่นได้

คำถามที่พบบ่อย

ถ. การโพสต์ของฉันล้มเหลว

โดยปกติหมายความว่าเครือข่ายสังคมได้เปลี่ยน DOM ของมัน ใช้ วินิจฉัย → รายงาน ใน popup การ hot-fix ตัวเลือกมักจะส่งภายในไม่กี่วันผ่านการอัปเดตอัตโนมัติของ selectors.json (ไม่ต้องอัปเดตส่วนขยาย)

ถ. Tutti ไม่จำการเข้าสู่ระบบของฉัน

Tutti พึ่งพาการเข้าสู่ระบบเบราว์เซอร์ปกติของเครือข่าย เข้าสู่ระบบแต่ละเครือข่ายใน Chrome ก่อน จากนั้น Tutti จะสามารถโพสต์ในนามของคุณได้ โหมดไม่ระบุตัวตนไม่ทำงานเพราะไม่มีเซสชันเครือข่ายอยู่ที่นั่น

ถ. ขีดจำกัดอักขระต่างกันในแต่ละเครือข่าย — เป็นปัญหาหรือไม่?

ไม่ใช่ โพสต์ยาวที่เกินขีดจำกัดอักขระของเครือข่ายจะถูกแบ่งเป็นเธรดต่อเนื่องโดยอัตโนมัติ (บน X เป็นห่วงโซ่ตอบกลับที่เหมาะสม; บนเครือข่ายอื่น ๆ เป็นโพสต์ตามลำดับ) โพสต์ภายในขีดจำกัดจะถูกส่งไปตามที่เป็น

ถ. ฉันได้รับการแจ้งว่าภาพ / วิดีโอของฉันใหญ่เกินไป

ภาพจะถูกปรับขนาดโดยอัตโนมัติผ่าน Canvas เพื่อให้พอดีกับขีดจำกัดขนาดของแต่ละเครือข่าย วิดีโอจะถูกบีบอัดโดยอัตโนมัติด้วย ffmpeg.wasm เมื่อเกินขีดจำกัดของเครือข่าย หากแม้แต่การบีบอัดก็ไม่สามารถใส่ได้ คุณจะเห็นข้อผิดพลาด — โปรดยืนยันก่อนว่าไฟล์อัปโหลดผ่านแอปพลิเคชันดั้งเดิมของเครือข่ายได้

ถ. Tutti เห็นเนื้อหาโพสต์ของฉันหรือไม่?

ไม่เห็น เนื้อหาโพสต์ไม่เคยผ่านเซิร์ฟเวอร์บุคคลที่สาม Tutti ฉีดเข้าโดยตรงไปยังหน้าสร้างโพสต์เว็บของแต่ละเครือข่าย ดูรายละเอียดในนโยบายความเป็นส่วนตัวของเรา ข้อยกเว้นเดียวคือเมื่อคุณกดปุ่มรายงานในผลิตภัณฑ์ ซึ่งส่งข้อมูลข้อผิดพลาดที่แก้ไขแล้วไปยังตัวติดตามประเด็นส่วนตัวของเรา

ถ. ฉันบริจาคได้ไหม?

ยังไม่ได้ — เรากำลังทำงานในช่องทางสนับสนุนที่ไม่กดดัน ค่าใช้จ่ายต่อเนื่องที่ใหญ่ที่สุดคือการตามทันการเปลี่ยนแปลง UI ของแต่ละเครือข่ายสังคม; กระแสเงิน Ko-fi ในอนาคตจะเป็นทุนสำหรับการบำรุงรักษานั้น
