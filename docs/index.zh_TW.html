Tutti

一個 Chrome 擴充功能。11 個社群網路。零麻煩。

GitHub 支援 隱私權政策

概觀

Tutti 一鍵將同一內容廣播到 11 個社群網路 (X、Bluesky、Threads、Tumblr、Mastodon、Misskey、Pixiv、TikTok、YouTube Shorts、Instagram、DeviantArt)。沒有後端伺服器,發布內容絕不會經過任何第三方伺服器。

一句話隱私說明

Tutti 沒有後端伺服器。發布內容 (文字、圖片、影片) 直接從您的瀏覽器送至您選擇的每個社群網路 — 絕不會透過任何開發者控制的伺服器。

完整政策: privacy.html
