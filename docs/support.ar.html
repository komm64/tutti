هل تحتاج إلى مساعدة؟

1. فشل النشر / شيء ما يتصرف بشكل غريب

استخدم زر «التشخيص» → «إبلاغ» داخل النافذة المنبثقة. هذه هي القناة الأساسية. يتم تنقيح التقرير (المستخدمين، عناوين URL، رسائل البريد الإلكتروني) وإعادة توجيهه إلى متتبع issue الخاص بنا، حيث يتم فتح PR لإصلاح المحدد تلقائيًا ودفعه كـ hot-fix لجميع المستخدمين في غضون أيام.

لماذا تستخدم زر التقرير داخل المنتج؟

يرفق تدفق التقرير في النافذة المنبثقة تلقائيًا حالة التأليف النهائية ولقطة DOM منقحة ونتائج تدقيق المحددات — بالضبط ما هو ضروري لإصلاح كسر المحدد بسرعة. يتم تجريد المعلومات الشخصية القابلة للتعريف عبر طبقات متعددة قبل الإرسال.

2. أسئلة عامة / طلبات ميزات

راسلنا عبر البريد الإلكتروني على contact@komm64.com. هذه قناة خاصة، آمنة لمشاركة لقطات الشاشة / المستخدمين الشخصيين.

3. أسئلة التثبيت / الحساب

Tutti يعيد استخدام تسجيلات دخول المتصفح الموجودة لديك لكل شبكة اجتماعية. لا يوجد حساب Tutti منفصل أو خطوة تسجيل دخول. بمجرد تسجيل الدخول إلى شبكة اجتماعية في نفس ملف تعريف Chrome، يمكن لـ Tutti النشر من هناك.

الأسئلة الشائعة

س. فشل منشوري.

يعني هذا عادةً أن الشبكة الاجتماعية غيرت DOM الخاص بها. استخدم التشخيص → إبلاغ في النافذة المنبثقة. عادةً ما يتم شحن hot-fix للمحدد في غضون أيام عبر التحديث التلقائي لـ selectors.json (لا حاجة لتحديث الإضافة).

س. لا يتذكر Tutti تسجيل الدخول الخاص بي.

يعتمد Tutti على تسجيل الدخول العادي للمتصفح للشبكة. سجّل الدخول إلى كل شبكة في Chrome أولاً، ثم يمكن لـ Tutti النشر نيابةً عنك. لا يعمل وضع التصفح المتخفي لأن جلسات الشبكة غير موجودة هناك.

س. تختلف حدود الأحرف لكل شبكة — هل هذا مشكلة؟

لا. يتم تقسيم المنشورات الطويلة التي تتجاوز حد أحرف الشبكة تلقائيًا إلى موضوع متابعة (في X، سلسلة ردود مناسبة؛ في الشبكات الأخرى، منشورات متتالية). تمر المنشورات ضمن الحد كما هي.

س. أُخبرت أن صورتي / فيديو الخاص بي كبير جدًا.

يتم تغيير حجم الصور تلقائيًا عبر Canvas لتناسب حد حجم كل شبكة. يتم ضغط مقاطع الفيديو تلقائيًا باستخدام ffmpeg.wasm عندما تكون فوق حد الشبكة. إذا لم يتناسب الضغط أيضًا، فسترى خطأً — يرجى التحقق أولاً من تحميل الملف عبر التطبيق الأصلي للشبكة.

س. هل يرى Tutti محتوى منشوري؟

لا. لا يمر محتوى المنشور أبدًا عبر خادم طرف ثالث. يحقن Tutti مباشرةً في صفحة التأليف الويب لكل شبكة. راجع سياسة الخصوصية الخاصة بنا للحصول على التفاصيل. الاستثناء الوحيد هو عند الضغط على زر التقرير داخل المنتج، الذي يرسل معلومات الخطأ المنقحة إلى متتبع issue الخاص بنا.

س. هل يمكنني التبرع؟

ليس بعد — نحن نعمل على قناة دعم منخفضة الضغط. أكبر تكلفة مستمرة هي مواكبة تغييرات واجهة المستخدم لكل شبكة اجتماعية؛ ستمول تدفقات Ko-fi المستقبلية تلك الصيانة.
