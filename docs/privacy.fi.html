Tutti — Tietosuojakäytäntö

Ei kolmansien osapuolten tietojen keräystä. Kaikki julkaisusisältö menee suoraan selaimestasi jokaiseen SNS:ään.

Etusivu GitHub Tuki

Päivitetty viimeksi: 2026-05-23 (v0.5.0)

1. Keräämämme ja siirtämämme tiedot

Tutilla ei ole taustapalvelinta. Julkaisusisältöä (teksti, kuvat, video) ei koskaan siirretä minkään kolmannen osapuolen palvelimelle. Se kulkee suoraan selaimestasi jokaiseen valitsemaasi sosiaaliseen verkostoon, aivan kuin olisit itse kirjoittanut ja lähettänyt sen siellä.

  • Julkaisun teksti, kuvat ja video välitetään vain valitsemillesi SNS-sivuille (X, Bluesky, Threads, Mastodon, Misskey, Tumblr, Pixiv, DeviantArt, Instagram, TikTok, YouTube) selaimesi sisällä.
  • Kun valinnainen virallinen API-reitti on käytössä Bluesky / Mastodon / Misskeyssä, julkaisut lähetetään suoraan selaimestasi kyseisen SNS:n omaan API-palvelimeen käyttäen Asetuksissa antamiasi tunnistetietoja. Mikään Tutti- / kehittäjäpalvelin ei ole mukana.
  • Mitään analytiikka-, seuranta- tai mainostietoja ei kerätä.
  • Tutti ei lue eikä tallenna todennustunnuksia, evästeitä tai istuntotietoja SNS-sivustoilta.

Poikkeus — manuaaliset virheraportit (suostumus napsautuksella)

Kun (ja vain kun) painat ”Raportoi”-painiketta, joka näytetään virhevalintaikkunassa tai diagnostiikkapaneelissa, seuraava lähetetään Cloudflare Workers -välityksen kautta yksityiseen GitHub-issue-trackeriin (komm64/tutti-issues, näkyy vain Tutti-kehittäjälle):

  • Virheilmoitus, laajennuksen versio, selaimen User-Agent-merkkijono.
  • Laajennuksen sisäisen lokin viimeiset 30 merkintää (määrittämäsi lokitaso; oletuksena INFO). Julkaisusisältöä, kuvia, videoita, SNS-kirjautumistietoja ja API-tunnistetietoja ei koskaan sisällytetä.
  • Toimitettu DOM-tilannekuva SNS-välilehdestä, jossa virhe tapahtui, käytetään rikkoutuneiden valitsimien korjaamiseen. Tekstisisältö, attribuuttiarvot, URL-polut/-kyselyt/-fragmentit, href/src-attribuutit ja <meta>/<link>-tunnisteet poistetaan. Hyötykuorman yläraja on 8 kt. Katso toteutus kohdasta dom-snapshot.ts.

Mitään ei lähetetä, ennen kuin painat raportointipainiketta. Kohde on yksityinen, näkyvissä vain Tutti-kehittäjälle.

2. Paikallisesti tallennetut tiedot

Seuraava tallennetaan vain selaimeesi Chromen storage-API:n kautta. Tutti ei siirrä mitään laitteen ulkopuolelle.

  • Asetukset (chrome.storage.sync): Mastodon-/Misskey-instanssin URL:t, autoPost-kytkin, näyttötila, valitsimen ohitus-URL jne. Voidaan replikoida muihin laitteisiisi Chrome Syncin kautta, jos olet kirjautuneena Chromeen.
  • Luonnokset (chrome.storage.session): Käynnissä oleva julkaisuteksti ja liitetyt mediat (kuvat / video). Tyhjennetään automaattisesti, kun selain suljetaan.
  • Valitut SNS:t (chrome.storage.local): SNS:t, jotka olet merkinnyt julkaisukohteiksi. Säilyy istuntojen välillä mukavuuden vuoksi; vain laitteen paikallisesti.
  • Viimeksi nähdyt käyttäjänimet (chrome.storage.local): Kuhunkin SNS:ään kirjautunut nykyinen käyttäjänimi, näytetään ponnahdusikkunassa, jotta voit vahvistaa, mikä tili vastaanottaa julkaisun. Vain laitteen paikallisesti.
  • Julkaisuhistoria (chrome.storage.local): Viimeiset 20 julkaisua: tekstin 80 ensimmäistä merkkiä, kohdealustat, onnistuminen/epäonnistuminen alustakohtaisesti ja aikaleima. Vain laitteen paikallisesti.
  • API-tunnistetiedot (chrome.storage.local): Vain jos otit nimenomaisesti käyttöön virallisen API-reitin Bluesky / Mastodon / Misskeylle: Bluesky app password tai Mastodon-/Misskey-käyttötunnus. Vain laitteen paikallisesti; ei koskaan replikoida Chrome Syncin kautta; ei koskaan siirretä mihinkään kehittäjäpalvelimelle. Poistettavissa Asetukset-velhosta.
  • Valitsimen ohituksen välimuisti (chrome.storage.local): Valitsimen pikakorjaustiedot, jotka on haettu Asetuksissa määrittämästäsi URL-osoitteesta (oletus: julkinen selectors.json Tutin GitHub Pagesissa).
  • Videon latausrajan välimuisti (chrome.storage.local): API-koetuksien tulos tilikohtaisille videon latausrajoille (esim. Bluesky). Vain numeeriset arvot; ei tunnistetietoja.

Kaikki tämä voidaan pyyhkiä milloin tahansa poistamalla laajennus asennuksesta tai tyhjentämällä Chromen laajennusmuisti.

3. Käytetyt luvat

LupaTarkoitus
storageTallenna yllä olevat paikalliset tiedot (asetukset, luonnokset, historia, valitut alustat jne.) selaimeesi.
offscreenSuorita ffmpeg.wasm näytön ulkopuolisessa asiakirjassa pakatakseen / uudelleen koodatakseen videotiedostoja, kun lähde ylittää kohde-SNS:n kokorajan. Videotietoja käsitellään vain paikallisesti; Tutti ei siirrä niitä minnekään.
sidePanelValinnainen UI-näyttötila: näyttää Tutin julkaisun laatimislomakkeen Chromen sivupaneelissa toimintoponnahdusikkunan sijaan, jotta se pysyy auki, kun vaihdat SNS-välilehtien välillä. Otat sen käyttöön Asetukset-sivulta; oletusarvo on tavallinen ponnahdusikkuna.
host_permissionsInjektoi sisältöskriptit kuhunkin SNS-sivustoon (x.com / twitter.com / bsky.app / bsky.social / threads.net / threads.com / mastodon.social / misskey.io / tumblr.com / pixiv.net / tiktok.com / youtube.com / instagram.com / deviantart.com) täyttääksesi julkaisun laatimislomakkeen julkaisusisällölläsi ja lähettääksesi sen puolestasi. Näiltä sivustoilta ei lueta tietoja ja lähetetä muualle; sisältöskripti kirjoittaa vain oman syötteesi.
optional_host_permissions: https://*/*Pyydetään ajon aikana vain, jos määrität mukautetun Mastodon- tai Misskey-instanssin URL:n (muu kuin oletusarvo). Chrome näyttää tavallisen lupakehotteen tuona hetkenä.

4. Suhde SNS-palveluntarjoajiin

Tutti automatisoi 11 tuetun verkoston web-UI:n (tai valinnaisesti julkaisee virallisen API:n kautta). Tämä on epävirallinen toiminto; Tutti ei ole sidoksissa X:ään, Blueskyyn, Metaan (Threads / Instagram), Mastodon gGmbH:hen, Misskeyhin, Tumblriin, Pixiviin, DeviantArtiin, TikTokiin tai YouTubeen. Olet edelleen sidottu kunkin palvelun omiin käyttöehtoihin.
