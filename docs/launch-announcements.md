# Beta launch announcement templates

Copy-paste templates for X / Discord / GitHub announcements. Drop in the
Web Store URL once the listing is live.

---

## X (English) — short, 280 chars max

```
🎉 Released Tutti (Beta) — a Chrome extension that cross-posts your text & images to X / Bluesky / Threads / Tumblr / Mastodon / Misskey from one popup.

No backend, no data leaves your browser. Preview-first by default.

📦 [Web Store URL]
🐛 [GitHub URL]
```
