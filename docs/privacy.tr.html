Tutti — Gizlilik politikası

Üçüncü taraf veri toplama yok. Tüm gönderi içeriği tarayıcınızdan doğrudan her SNS'ye gider.

Ana sayfa GitHub Destek

Son güncelleme: 2026-05-23 (v0.5.0)

1. Topladığımız ve ilettiğimiz veriler

Tutti'nin arka uç sunucusu yoktur. Gönderi içeriği (metin, resimler, video) asla üçüncü taraf bir sunucuya iletilmez. Tıpkı oraya kendiniz yazıp göndermişsiniz gibi, doğrudan tarayıcınızdan seçtiğiniz her sosyal ağa gider.

  • Gönderi metni, resimleri ve videosu yalnızca tarayıcınız içinde seçtiğiniz SNS sayfalarına (X, Bluesky, Threads, Mastodon, Misskey, Tumblr, Pixiv, DeviantArt, Instagram, TikTok, YouTube) iletilir.
  • Bluesky / Mastodon / Misskey için isteğe bağlı resmi API yolu etkinleştirildiğinde gönderiler, Ayarlar'da sağladığınız kimlik bilgileri kullanılarak doğrudan tarayıcınızdan o SNS'nin kendi API sunucusuna gönderilir. Hiçbir Tutti / geliştirici sunucusu dahil değildir.
  • Hiçbir analitik, izleme veya reklam verisi toplanmaz.
  • Tutti, SNS sitelerinden kimlik doğrulama belirteçlerinizi, çerezlerinizi veya oturum verilerinizi okumaz ve depolamaz.

İstisna — manuel hata raporları (her tıklamada onay)

Bir hata iletişim kutusunda veya tanılama panelinde gösterilen “Bildir” düğmesine bastığınızda (ve yalnızca o zaman), aşağıdakiler Cloudflare Workers aktarımı üzerinden özel bir GitHub konu izleyicisine (komm64/tutti-issues, yalnızca Tutti geliştiricisine görünür) gönderilir:

  • Hata mesajı, uzantı sürümü, tarayıcı User-Agent dizesi.
  • Uzantının dahili günlüğünün son 30 girişi (yapılandırdığınız günlük düzeyi; varsayılan INFO). Gönderi içeriği, resimler, video, SNS oturum açma verileri ve API kimlik bilgileri asla dahil edilmez.
  • Hatanın oluştuğu SNS sekmesinin redakte edilmiş DOM anlık görüntüsü, bozuk seçicileri onarmak için kullanılır. Metin içeriği, öznitelik değerleri, URL yolları/sorguları/parçaları, href/src öznitelikleri ve <meta>/<link> etiketleri çıkarılır. Yük 8 KB ile sınırlıdır. Uygulama için dom-snapshot.ts bakın.

Raporlama düğmesine basana kadar hiçbir şey gönderilmez. Hedef özeldir, yalnızca Tutti geliştiricisi görebilir.

2. Yerel olarak depolanan veriler

Aşağıdakiler yalnızca tarayıcınızda Chrome'un storage API'si aracılığıyla saklanır. Tutti hiçbir şeyi cihaz dışına aktarmaz.

  • Ayarlar (chrome.storage.sync): Mastodon / Misskey örnek URL'leri, autoPost anahtarı, görüntüleme modu, seçici geçersiz kılma URL'si vb. Chrome'da oturum açtıysanız Chrome Sync aracılığıyla diğer cihazlarınıza kopyalanabilir.
  • Taslaklar (chrome.storage.session): Süren gönderi metni ve eklenmiş medya (resim / video). Tarayıcı çıkışında otomatik olarak temizlenir.
  • Seçilen SNS (chrome.storage.local): Gönderi hedefleri olarak işaretlediğiniz SNS'ler. Kolaylık için oturumlar arasında kalıcıdır; yalnızca cihaza yerel.
  • Son görülen kullanıcı adları (chrome.storage.local): Her SNS'de şu anda oturum açmış olan kullanıcı adı, gönderiyi hangi hesabın alacağını onaylayabilmeniz için popup'ta gösterilir. Yalnızca cihaza yerel.
  • Gönderi geçmişi (chrome.storage.local): Son 20 gönderi: metnin ilk 80 karakteri, hedef platformlar, platform başına başarı/başarısızlık ve zaman damgası. Yalnızca cihaza yerel.
  • API kimlik bilgileri (chrome.storage.local): Yalnızca Bluesky / Mastodon / Misskey için resmi API yolunu açıkça etkinleştirdiyseniz: Bluesky app password veya Mastodon / Misskey erişim belirteci. Yalnızca cihaza yerel; asla Chrome Sync üzerinden çoğaltılmaz; asla herhangi bir geliştirici sunucusuna iletilmez. Ayarlar sihirbazından kaldırılabilir.
  • Seçici geçersiz kılma önbelleği (chrome.storage.local): Ayarlar'da yapılandırdığınız URL'den alınan seçici sıcak düzeltme verileri (varsayılan: Tutti'nin GitHub Pages'taki genel selectors.json).
  • Video yükleme sınırı önbelleği (chrome.storage.local): Hesap başına video yükleme sınırları için API probelerinin sonucu (ör. Bluesky). Yalnızca sayısal değerler; kimlik bilgisi yok.

Tüm bunlar uzantıyı kaldırarak veya Chrome'un uzantı depolamasını temizleyerek istediğiniz zaman silinebilir.

3. Kullanılan izinler

İzinAmaç
storageYukarıdaki yerel verileri (ayarlar, taslaklar, geçmiş, seçili platformlar vb.) tarayıcınızda depolamak için.
offscreenKaynak, hedef SNS'nin boyut sınırını aştığında ffmpeg.wasm'ı offscreen belgesinde çalıştırarak video dosyalarını sıkıştırmak / yeniden kodlamak için. Video verileri yalnızca yerel olarak işlenir; Tutti tarafından hiçbir yere aktarılmaz.
sidePanelİsteğe bağlı UI görüntüleme modu: Tutti'nin yazma formunu eylem popup'ı yerine Chrome'un yan panelinde gösterir, böylece SNS sekmeleri arasında geçiş yaparken açık kalır. Seçenekler sayfasından etkinleştirirsiniz; varsayılan standart popup'tır.
host_permissionsHer SNS sitesine (x.com / twitter.com / bsky.app / bsky.social / threads.net / threads.com / mastodon.social / misskey.io / tumblr.com / pixiv.net / tiktok.com / youtube.com / instagram.com / deviantart.com) content scriptleri enjekte ederek yazma formunu gönderi içeriğinizle doldurmak ve sizin adınıza göndermek için. Bu sitelerden hiçbir veri okunmaz ve başka bir yere gönderilmez; content script yalnızca kendi girdinizi yazar.
optional_host_permissions: https://*/*Yalnızca özel bir Mastodon veya Misskey örnek URL'si yapılandırdığınızda (varsayılan dışında) çalışma zamanında istenir. Chrome o anda standart izin istemini gösterir.

4. SNS sağlayıcıları ile ilişki

Tutti, desteklenen 11 ağın web UI'sını otomatikleştirir (veya isteğe bağlı olarak resmi API üzerinden yayınlar). Bu resmi olmayan bir işlevdir; Tutti X, Bluesky, Meta (Threads / Instagram), Mastodon gGmbH, Misskey, Tumblr, Pixiv, DeviantArt, TikTok veya YouTube ile bağlantılı değildir. Her hizmetin kendi kullanım koşullarına bağlı kalmaya devam edersiniz.
