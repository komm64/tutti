Tutti

1 つの Chrome 拡張で 11 の SNS にクロスポスト。手間ゼロ。

GitHub サポート プライバシーポリシー

概要

Tutti は同じ投稿を 11 のソーシャルネットワーク (X, Bluesky, Threads, Tumblr, Mastodon, Misskey, Pixiv, TikTok, YouTube Shorts, Instagram, DeviantArt) にワンクリックで一斉送信します。バックエンドを持たず、投稿内容はいかなる第三者サーバーも経由しません。

プライバシーを一言で

Tutti にバックエンドサーバーは無く、投稿コンテンツ (テキスト・画像・動画) はブラウザから選択した各 SNS へ直接送信されます。開発者の管理サーバーを経由することはありません。

完全なポリシー: privacy.html
