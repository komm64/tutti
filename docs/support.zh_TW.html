需要協助嗎?

1. 發布失敗 / 行為異常

請使用 popup 中的「診斷」→「回報」按鈕。這是正規管道。回報經過 redact (使用者名稱、URL、電子郵件) 後轉送至我們的私有 issue tracker,在那裡會自動建立 selector 修復 PR,數日內以熱修復形式推送至全體使用者。

為什麼使用產品內的回報按鈕?

popup 的回報流程會自動附上最終發布狀態、redacted DOM 快照及選擇器稽核結果 — 這正是快速修復 selector 失效所需的內容。個人識別資訊在傳送前經多層 redact。

2. 一般問題 / 功能請求

請寄信至 contact@komm64.com。這是私有管道,可安全分享螢幕截圖 / 個人使用者名稱。

3. 安裝 / 帳號問題

Tutti 重複使用您各個社群網路的現有瀏覽器登入。無需單獨的 Tutti 帳號或登入步驟。只要您在同一個 Chrome 設定檔中已登入某個社群網路,Tutti 即可從該處發布。

FAQ

Q. 我的發布失敗了。

通常表示某個社群網路更改了其 DOM。請使用 popup 中的 診斷 → 回報。選擇器熱修復通常會在數日內透過 selectors.json 自動更新發布 (無需更新擴充功能)。

Q. Tutti 記不住我的登入。

Tutti 依賴網路的一般瀏覽器登入。請先在 Chrome 中登入每個網路,然後 Tutti 才能代您發布。無痕模式無法運作,因為該處沒有網路的工作階段。

Q. 各網路的字元上限不同 — 這有問題嗎?

沒有問題。超過某網路字元上限的長文會自動拆分為連續推文 (在 X 上為真正的回覆鏈;其他網路為連續發布)。在上限以內的發布則原樣送出。

Q. 系統提示我的圖片 / 影片過大。

圖片透過 Canvas 自動調整大小以符合各網路的大小上限。影片在超過網路上限時透過 ffmpeg.wasm 自動壓縮。若壓縮後仍無法符合,您將看到錯誤 — 請先透過該網路的原生應用程式確認檔案能上傳。

Q. Tutti 能看到我的發布內容嗎?

不能。發布內容絕不會經過任何第三方伺服器。Tutti 直接注入各網路的 Web 發布頁面。詳情請參閱我們的隱私權政策。唯一例外是您按下產品內回報按鈕時,這會將 redacted 錯誤資訊傳送至我們的私有 issue tracker。

Q. 我可以捐款嗎?

暫時還不行 — 我們正在準備低壓力的支援管道。最大的持續成本是跟進每個社群網路的 UI 變更;未來的 Ko-fi 流程將用於資助這種維護。
