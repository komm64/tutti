GitHub プライバシーポリシー Privacy Policy (EN)

Overview

Tutti は X / Bluesky / Threads / Mastodon / Misskey に同じ投稿を一気に流す Chrome 拡張です。バックエンドを持たず、投稿内容は第三者サーバーを一切経由しません。

プライバシーポリシー

最終更新: 2026-04-30

1. 取得・送信するデータ

本拡張はいかなる第三者サーバーにもデータを送信しません。

  • 投稿テキスト・画像・動画は、ユーザーが選択した各 SNS のページに対し、ブラウザ内でのみ渡されます。
  • SNS アカウント情報・認証トークン・Cookie 等を本拡張が取得・保存・送信することはありません。
  • 解析・トラッキング・広告のためのデータ収集は行いません。

2. ローカルに保存するデータ

Chrome の storage API でユーザーのブラウザ内にのみ保存します。

  • 設定(chrome.storage.sync): Mastodon / Misskey インスタンス URL 等。Chrome 同期で他デバイスに複製される場合あり
  • 投稿履歴(chrome.storage.local): 直近 20 件の先頭 80 文字、対象プラットフォーム、成否、タイムスタンプ。デバイスローカルのみ
  • ログイン中アカウント情報(chrome.storage.local): 各 SNS の現在ログイン中ユーザー名(誤爆防止のため popup に表示)。デバイスローカルのみ
  • 下書き(chrome.storage.session): 入力中のテキストとプラットフォーム選択。ブラウザ終了で自動消去

これらは拡張のアンインストールでいつでも完全削除できます。

3. 必要な権限

権限用途
storage上記データのローカル保存
offscreen動画整形を行う offscreen document の生成 (将来機能)
host_permissionsx.com / bsky.app / threads.com / mastodon.social / misskey.io 等の投稿ページにアクセス
optional_host_permissions: https://*/*ユーザーが Mastodon / Misskey の代替インスタンスを設定した際にのみ動的に要求

4. 外部サービスとの関係

Tutti は X・Bluesky・Threads・Mastodon・Misskey の各 Web 画面を自動操作しますが、 これらは公式機能ではなく、各社・運営とは無関係です。各サービスの利用規約は 各サービスのものに従ってください。

Privacy Policy (English)

Last updated: 2026-04-30

1. Data collected and transmitted

This extension never transmits data to any third-party server.

  • Post text, images, and videos are passed only to the user-selected SNS pages within the browser.
  • The extension does not retrieve, store, or transmit SNS account credentials, authentication tokens, or cookies.
  • No analytics, tracking, or advertising data is collected.

2. Locally stored data

Stored only inside the user's browser via Chrome's storage API.

  • Settings (chrome.storage.sync): Mastodon / Misskey instance URLs. May be replicated to other devices via Chrome Sync.
  • Post history (chrome.storage.local): Last 20 entries (first 80 chars of text, target platforms, success flags, timestamps). Local only.
  • Logged-in account names (chrome.storage.local): Current account on each SNS (for safety display in popup). Local only.
  • Drafts (chrome.storage.session): In-progress text and platform selection. Auto-cleared on browser exit.

All of the above can be erased at any time by uninstalling the extension.

4. Relationship with external services

Tutti automates the web UIs of X, Bluesky, Threads, Mastodon, and Misskey, but is not affiliated with any of them. Use of each service is subject to that service's own terms.
