Tutti позволяет написать один раз и разослать тот же пост во все ваши социальные сети одним кликом (поддерживается 11 сетей). Текст, превышающий лимит, автоматически разбивается (X использует правильную цепочку ответов, так что это становится thread); изображения автоматически изменяются под ограничения каждой платформы; видео проверяются по длительности / размеру, а слишком большие клипы перекодируются на лету с помощью `ffmpeg.wasm`.

**Содержимое ваших постов никогда не касается какого-либо стороннего сервера.**

🔒 [Политика конфиденциальности](https://komm64.github.io/tutti/)

## Возможности

- 📤 **Рассылка по множеству сетей** — напишите один раз, кликните один раз, опубликуйте в каждую выбранную сеть (11 сетей)
- ✂️ **Автоматическое разделение текста сверх лимита** — пронумерованы как `(1/N)`, публикуются последовательно. На X они соединяются как **цепочка ответов (thread)**, в других сетях публикуются независимо
- Границы `#hashtag` сохраняются при разделении / Bluesky получает правильные **rich-text facets** (кликабельные теги + URL annotations)
- 🖼️ **До 4 изображений + автоматическое изменение размера** — автоматически вписывается в строгие ограничения, такие как лимит 1 МБ Bluesky
- 🎬 **Публикация видео + автоматическое сжатие** — клипы сверх лимита перекодируются на месте `ffmpeg.wasm` (в офскрин-документе)
- 🔌 **Опциональный путь через официальный API** — для Bluesky / Mastodon / Misskey зарегистрируйте учётные данные в Настройках, и Tutti публикует через публичный API вместо DOM-скриптинга (устойчив к изменениям UI SNS)
- 📊 **Прогресс в реальном времени** — видите статус каждой сети в реальном времени
- 🪪 **Отображение залогиненного аккаунта** — попап показывает, с какого аккаунта будет публиковать каждая сеть (помогает предотвратить случайности)
- 🛡️ **Переключатель autoPost** — выключен по умолчанию. Режим по умолчанию открывает каждую страницу создания, заполняет тело + вложения и **останавливается прямо перед нажатием кнопки публикации** (режим "предпросмотра"), чтобы вы могли заметить ошибки
- 📜 **История постов** — последние 20 записей сохранены локально
- 💾 **Автосохранение черновиков** — ваш текст переживает закрытие попапа
- ⌨️ **Ctrl/Cmd + Enter для публикации**
- ⚙️ **Переключение инстансов Mastodon / Misskey** — укажите на любой инстанс из Настроек
- 🩹 **Hot-fix селекторов** — когда DOM SNS меняется и ломает путь, Tutti может загрузить патч `selectors.json`, чтобы вам не нужно было ждать следующего релиза расширения
- 🐞 **Кнопка отчёта об ошибках** — один клик из попапа отправляет GitHub issue с отредактированным DOM-снимком (auto-triage пайплайн превращает это в PR селектора)
- 🌐 **Локализован** — 31 язык (попап + опции)

## Поддерживаемые сети

11 сетей. "Stable" означает, что реальная публикация проверена end-to-end; "Experimental" означает, что адаптер подключен, но реальная публикация с autoPost ещё не полностью валидирована. Для Experimental начинайте в режиме предпросмотра (autoPost ВЫКЛ).

### Stable (реальная публикация проверена)

| Сеть | text | image | shortVideo | longVideo | Путь |
|---|:---:|:---:|:---:|:---:|---|
| X | ✅ | ✅ | ✅ | ✅ | DOM |
| Bluesky | ✅ | ✅ | ✅ | — | DOM + API |
| Threads | ✅ | ✅ | ✅ | ✅ | DOM |
| Mastodon | ✅ | ✅ | ✅ | ✅ | DOM + API |
| Misskey | ✅ | ✅ | ✅ | ✅ | DOM + API |
| Tumblr | ✅ | ✅ | ✅ | ✅ | DOM |
| Pixiv | — | ✅ | — | — | DOM (multi-step) |
| TikTok | — | — | ✅ | — | DOM (multi-step) |
| YouTube (Shorts) | — | — | ✅ | — | DOM (multi-step) |
| Instagram | — | ✅ | ✅ | — | DOM (multi-step) |

### Experimental (только адаптер; реальная публикация с autoPost ещё не проверена)

| Сеть | text | image | shortVideo | longVideo | Путь |
|---|:---:|:---:|:---:|:---:|---|
| DeviantArt | — | ✅ | — | — | DOM (multi-step) |

- **DOM**: Tutti автоматизирует веб-UI создания SNS (более чувствительный к анти-бот изменениям)
- **DOM + API**: Если вы сохраните учётные данные в Настройках, Tutti переключится на официальный API. При сбое API Tutti **не возвращается тихо к DOM** — вы увидите явную ошибку. Без учётных данных запускается только DOM-путь.
- **multi-step**: Для модальных окон в стиле мастера на нескольких шагах (фреймворк: `executeMultiStepFlow`)

## Установка

### Chrome Web Store

Опубликовано (Unlisted): [Tutti в Web Store](https://chromewebstore.google.com/detail/tutti/mcjfgdcffjfhkcepfpnifcpknlddmbpe)

### Распакованная / dev-сборка

Скачайте последний zip из [Releases](https://github.com/komm64/tutti/releases), затем:

1. Распакуйте его
2. Откройте `chrome://extensions/` (или `brave://extensions/` в Brave)
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение" и выберите распакованную папку

## Конфиденциальность

Текст постов, изображения и видео обрабатываются **полностью внутри вашего браузера** — они никогда не отправляются ни на какой сторонний сервер. См. [политику конфиденциальности](https://komm64.github.io/tutti/) для деталей.
