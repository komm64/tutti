# Tutti プライバシーポリシー

最終更新日: 2026-04-29

Tutti(以下「本拡張」)は、ユーザーが入力したコンテンツを複数の SNS に
同時投稿することを目的とした Chrome 拡張機能です。本拡張の開発者(以下
「開発者」)は、ユーザーのプライバシーを尊重し、以下の方針でデータを
扱います。

## 1. 取得・送信するデータ

**本拡張は、いかなる第三者サーバーにもデータを送信しません。**

- 投稿テキスト・画像・動画は、ユーザーが選択した各 SNS(X / Bluesky /
  Threads / Mastodon)のページに対して、ユーザーのブラウザ内でのみ
  渡されます。
- ユーザーの SNS アカウント情報、認証トークン、Cookie 等を本拡張が
  取得・保存・送信することはありません。
- 解析(アナリティクス)・トラッキング・広告のためのデータ収集は
  一切行いません。

## 2. ローカルに保存するデータ

本拡張は、Chrome の `storage` API を用いて以下のデータをユーザーの
ブラウザ内にのみ保存します:

- **設定(`chrome.storage.sync`)**: Mastodon インスタンス URL 等の
  ユーザー設定。Google アカウントで Chrome にサインインしている場合、
  Chrome 同期によって他のデバイスにも複製されます。
- **投稿履歴(`chrome.storage.local`)**: 直近 20 件の投稿の先頭 80 文字、
  対象プラットフォーム、成否、タイムスタンプ。このデバイスにのみ保存され、
  外部に送信されることはありません。

これらのデータは、ユーザーが本拡張をアンインストールするか、Chrome の
拡張機能データを削除することでいつでも完全に消去できます。

## 3. 必要な権限

| 権限 | 用途 |
|---|---|
| `storage` | 上記の設定・履歴をローカル保存するため |
| `offscreen` | 動画整形を行う offscreen document を生成するため(P7 以降) |
| `host_permissions: x.com / bsky.app / threads.net / mastodon.social` | 各 SNS の投稿ページを開きフォーム入力するため |
| `optional_host_permissions: https://*/*` | ユーザーが mastodon.social 以外のインスタンスを設定した場合にのみ、そのドメインへのアクセスを動的に求めるため |

## 4. 外部サービスとの関係

本拡張は X・Bluesky・Threads・Mastodon の各 Web 画面を自動操作しますが、
これらサービスの公式機能ではなく、各社とは無関係です。各サービスの利用規約
は各サービスのものに従ってください。
